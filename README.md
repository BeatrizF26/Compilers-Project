# CompSh

## How to run a `.csh` program

1. Run `antlr4-clean` (just to prevent issues)
2. Delete `Output.java` if it exists
3. Run `./build.sh`
4. Then `./compile.sh ../examples/des03.csh` *(example file)*
5. Finally `./run.sh Output.java`
